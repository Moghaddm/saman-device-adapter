🚀 DEPLOYMENT WORKFLOW
═════════════════════════════════════════════════════════════════════════════

1. BUILD
   ├─ Open SamanDeviceAdapter.csproj in Visual Studio
   ├─ Right-click → Properties → Build
   ├─ Verify Target Framework: .NET Framework 4.8
   ├─ Build → Build Solution (Release)
   └─ Output: bin\Release\SamanDeviceAdapter.exe

2. PREPARE
   ├─ Create deployment folder
   ├─ Copy SamanDeviceAdapter.exe
   ├─ Copy install.bat
   ├─ Copy uninstall.bat
   └─ Verify all 3 files are together

3. INSTALL (Run as Administrator)
   ├─ Right-click install.bat → Run as Administrator
   ├─ Script checks admin privileges
   ├─ Installs Windows Service
   ├─ Starts the service
   └─ Opens http://localhost:8080 in browser

4. CONFIGURE
   ├─ Dashboard opens automatically
   ├─ Enter Device IP (e.g., 192.168.1.201)
   ├─ Enter Device Port (default: 4370)
   ├─ Click "Test Connection"
   ├─ Set Company ID if needed
   ├─ Enable/disable "Delete after send"
   └─ Click "Save Settings"

5. MONITOR
   ├─ Dashboard refreshes every 5 seconds
   ├─ View Service Status (Green = running)
   ├─ View Device Connection (Green = connected)
   ├─ View sync history (last 20 operations)
   ├─ Check logs: ServiceLog_YYYY-MM-DD.txt
   └─ Verify in Services.msc

6. UNINSTALL (Run as Administrator)
   ├─ Right-click uninstall.bat → Run as Administrator
   ├─ Service stops automatically
   ├─ Service unregistered
   └─ Complete removal
